## 既然语言都有对应的数据结构和 API，为什么需要手撸代码？

* 算法问题与语言无关
* 语言提供的 API 是有限的，当有新的功能是 API 不提供的，就需要改写
* 任何软件工具的底层都是最基本的算法和数据结构
